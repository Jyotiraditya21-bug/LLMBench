Product Guide & How to Use

Welcome to LLMBench

LLMBench is a premium engineering dashboard for evaluating Large Language Models. Follow these simple steps to benchmark and audit prompts:

  1. Manage Datasets: Go to the Datasets tab to create evaluation folders, batch import test cases, or inspect stored ground truths.
  2. Trigger Evaluations: In the Evaluation Hub, select your target dataset and prompt version, select the model suites (e.g. GPT-4, Claude, Gemini), and start the execution queue. The pipeline runs asynchronously in Celery and will poll live metrics.
  3. Prompt Arena Benchmarking: Navigate to the Prompt Arena, select your Prompt Template Group, and load your baseline (A) and comparison (B) prompt runs to view full, multi-dimensional accuracy, latency, and cost regression deltas.
  4. Failure Diagnostics: Use the Agent RCA tab to aggregate failed evaluation outputs, allowing LLM reasoning agents to pinpoint failure modes and suggest prompt scaffolds.
